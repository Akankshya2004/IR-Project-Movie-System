# User Evaluation Tasks

## Instructions for Participants

Thank you for participating in our Movie IR System evaluation! 

You will complete 5 search tasks using the movie search system. For each task:
- Read the task description carefully
- Use the search system to find the answer
- Tell the researcher when you've completed the task
- Answer brief questions about the task

Please think aloud as you work - tell us what you're thinking and why you're making each choice.

---

## Task 1: Simple Known-Item Search

**Task**: Find the movie "Inception" and note its rating.

**Success Criteria**: 
- Found the correct movie
- Correctly noted the rating

**What to record**:
- Did you find the movie? (Yes/No)
- What rating did it have?
- How confident are you that this is correct? (1-5)

---

## Task 2: Faceted Search Task

**Task**: Find a highly-rated (8.0+) science fiction movie released after 2010.

**Success Criteria**:
- Movie is in the Sci-Fi genre
- Rating is 8.0 or higher
- Released in 2011 or later

**What to record**:
- Which movie did you choose?
- What is its rating and year?
- How difficult was this task? (1=very easy, 5=very difficult)
- How satisfied are you with your result? (1-5)

---

## Task 3: Genre Exploration

**Task**: Find 3 comedy movies from the 1990s with ratings above 7.0.

**Success Criteria**:
- All 3 movies are comedies
- All released between 1990-1999
- All have ratings > 7.0

**What to record**:
- List the 3 movies you found
- How difficult was this task? (1-5)
- How many searches did it take?
- Did you use any filters? Which ones?

---

## Task 4: Discovery Task

**Task**: Find a movie similar to "The Dark Knight" that you haven't seen before.

**Success Criteria**:
- Found a movie you haven't seen
- Movie is reasonably similar (action, superhero, or thriller)

**What to record**:
- Which movie did you find?
- How did you discover it?
- Would you actually be interested in watching it? (1-5)
- How satisfied are you with this recommendation? (1-5)

---

## Post-Task Questions

After completing all tasks, please answer:

### System Usability
1. How easy was the system to use? (1=very difficult, 5=very easy)
2. How quickly did you learn to use the system? (1=very slow, 5=very fast)
3. Did you encounter any problems? (Open-ended)

### Feature Usefulness (if applicable)
For users with full system access:

4. How useful were the genre filters? (1=not useful, 5=very useful)
5. How useful were the year filters? (1-5)
6. How useful were the rating filters? (1-5)
7. How useful was the "Find Similar Movies" feature? (1-5)
8. How useful was the sorting feature? (1-5)

### Overall Satisfaction
9. Overall, how satisfied are you with the system? (1=very dissatisfied, 5=very satisfied)
10. Would you use this system to find movies? (Yes/No/Maybe)
11. What did you like most about the system?
12. What would you improve?

### Search Behavior
13. Did you prefer keyword search or browsing with filters?
14. Did the results match what you were looking for?
15. Were the movie descriptions helpful?
